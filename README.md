# 图片压缩 GUI

一个本地运行的 PyQt 图片压缩工具，支持拖拽图片、设置目标大小、查看压缩进度、直观对比压缩效果并另存为文件。

## 安装依赖

如果你的 `python3.11` 在 PATH 里：

```bash
python3.11 -m pip install -r requirements.txt
```

如果命令找不到，用本机已检测到的路径：

```bash
/usr/local/bin/python3.11 -m pip install -r requirements.txt
```

## 运行

已打包版本可以从 GitHub Releases 下载，解压后直接双击 `图片压缩工具.app`。

源码版本可以运行：

```bash
/usr/local/bin/python3.11 app.py
```

## 使用方式

1. 拖入图片，或点击“选择图片”。
2. 设置目标大小，比如 `300 KB` 或 `1.5 MB`。
3. 在“导出格式”里选择或搜索格式，比如 `jpg`、`png`、`webp`、`透明`、`无损`。
4. 点击“开始压缩”。
5. 通过进度条查看当前压缩步骤和完成百分比。
6. 使用“并排对比”“滑动对比”“叠加对比”查看原图和压缩后的差异。
7. 使用“适应窗口”“100%”“缩小”“放大”调整预览清晰度；在非滑动模式下，放大后可以拖动画面查看细节。
8. 点击“下载 / 另存为”保存压缩后的图片。

说明：目标大小是近似匹配。程序会先调整 JPG/WebP 质量，如果仍然过大，会自动缩小图片尺寸。

当前支持的导出格式会根据本机 Pillow 能力自动显示，常见包括 JPG、PNG、WebP、BMP、TIFF、GIF、JPEG 2000、ICO。
